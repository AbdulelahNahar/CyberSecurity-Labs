# Week 1 – PreSecurity: Introduction Module

## Overview
This module introduces the fundamentals of cybersecurity, focusing on pre-security concepts and an overview of offensive and defensive security.

---

## Topics Covered
- **Cybersecurity Overview**: Importance of cybersecurity in IT.
- **Offensive Security**: Finding and exploiting vulnerabilities.
- **Defensive Security**: Protecting systems from attacks.
- **Penetration Testing**: Understanding the methodology of ethical hacking.
- **Career Paths**: Roles in offensive and defensive security.

---

## Key Learnings
- Offensive security focuses on identifying and exploiting weaknesses in systems.  
- Defensive security protects networks, systems, and data from attacks.  
- Penetration testing is a critical skill for offensive security professionals.  
- Pre-security modules help build a foundation before hands-on labs.
